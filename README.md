Overview

The Dijkstra's Algorithm Visualizer is an interactive project that demonstrates how Dijkstra’s shortest path algorithm works step by step. It helps users understand how nodes are explored, distances are updated, and the shortest path is calculated in a weighted graph.

This project is useful for students and beginners who want a clear understanding of shortest-path algorithms through visualization.

Features

Step-by-step visualization of Dijkstra’s Algorithm 

Interactive grid or graph-based interface

Option to add obstacles or weights

Real-time shortest path calculation

Simple and user-friendly interface

How Dijkstra’s Algorithm Works

Dijkstra’s Algorithm finds the shortest distance from a source node to all other nodes in a weighted graph with non-negative weights.

Steps:

1.Assign distance 0 to the source node and infinity to all other nodes.

2.Select the unvisited node with the smallest distance.

3.Update distances of its neighboring nodes.

4.Mark the current node as visited.

5.Repeat until all nodes are processed or the destination is reached.
